## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar os principais fundamentos da linguagem **PHP**, integrando lógica de programação com estrutura HTML e estilização CSS.

A aplicação exibe os resultados diretamente na tela, organizados visualmente em blocos individuais para cada exercício.

---

## 🎯 Objetivo

Demonstrar conhecimento prático em:

- Variáveis
- Operações matemáticas
- Estruturas condicionais (`if / else`)
- Estrutura `switch case`
- Laços de repetição:
  - `for`
  - `while`
  - `do while`
- Arrays simples
- Arrays associativos
- Integração entre PHP + HTML + CSS

---

## 🛠 Tecnologias Utilizadas

- **PHP**
- **HTML5**
- **CSS3**

---

## 📂 Estrutura do Projeto

O projeto é composto por um único arquivo `.php` contendo:

- Estrutura base HTML
- Estilização interna com CSS
- Blocos organizados por exercício
- Separação visual com:
  - `<div class="exercicio">`
  - `<div class="resultado">`
